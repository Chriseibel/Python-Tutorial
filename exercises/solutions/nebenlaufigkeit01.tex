Zur Verwendung von Threads muss das \lstinline$threading$-Modul importiert werden.
Zur Verwendung von \lstinline$sleep()$ muss das \lstinline$time$-Modul importiert werden.
Es muss eine Klasse von \lstinline$Thread$ abgeleitet werden.
Diese Klasse muss die \lstinline$run()$-Methode überschreiben und in einer \lstinline$for$-Schleife, 
welche von 1 bis 10 läuft, die aktuelle Zahle ausgeben und eine Sekunde ''schlafen''.

Es muss ein Objekt diese Klasse erzeugt werden und die \lstinline$start()$ ausgeführt werden.
