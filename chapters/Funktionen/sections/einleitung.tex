In diesem Kapitel gehen wir auf die Nutzung von Funktionen in Python ein.
Eine Funktion bildet einen Code-Block ab, also eine Sequenz von Befehlen, 
welche eine bestimmte \textbf{Funktion} erfüllt. 


Dieser Code-Block wird mit dem Schlüsselwort \textit{def} gestartet,
gefolgt vom Namen der Funktion, anschließend von Klammern
(welche Input-Parameter beinhalten können) und zum Schluss der Funktionsdefinition
folgt noch ein Doppelpunkt. 
Nach dem Doppelpunkt kommt die Befehlssequenz. Soll die Funktion
Werte zurück liefern, dann steht am Ende der Sequenz das
\textit{return}-Schlüsselwort. Achtung, eine Funktion liefert immer
einen Wert zurück, wird keiner angegeben, so wird der Wert \textit{None}
als Rückgabewert festgelegt.


\begin{lstlisting}[caption=Definition einer Funktion, label=funcDef]
def funktionsName (parameter):
  ...
  return rueckgabeWert
\end{lstlisting}
