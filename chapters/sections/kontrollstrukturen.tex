\section{Kontrollstrukturen}

Die Kontrollstrukturen in Python haben einen formalen Unterschied zu Java oder C++, funktional allerdings sind sie identisch. In Python werden keine geschweiften Klammern genutzt, um die Blöcke der einzelnen Abfragen abzugrenzen. Dazu genügt das Einrücken der Anweisung. Dies gilt sowohl für if-then-else-Bedingungen und Conditional Expressions, als auch für Schleifen. Im Folgenden schauen wir uns die einzelnen Strukturen im Detail und mit Beispielen an.

\subsection{If-then-else}

Die if-then-else-Struktur ermöglicht es, wie wir es bereits kennen, simple wenn-dann Abfragen zu tätigen.

Mehrere Abfragemöglichkeiten werden mit elif markiert. Vergleiche hierzu Listing \ref{refif}.

\begin{lstlisting}[caption={If-then-else},label=refif]
if statement1:
	print("Fall 1")
elif statement2:
	print("Fall 2")
else:
	print("Fall 3")

\end{lstlisting}

\textbf{Conditional Expressions}

Die Conditional Expressions (engl. bedingte Ausdrücke) stellen eine kompaktere Schreibweise als if-then-else-Bedinungen dar. (vgl. Listing \ref{refcond})

\begin{lstlisting}[caption={Conditional Expressions},label=refcond]
// Klassisches If-Else
if wort == "start":
	x = "los"
else:
	x = halt"
	
// If-Else als Conditional Expression
x = ("los" if wort == "start" else "halt")

\end{lstlisting}

\subsection{Schleifen}

Python hat sowohl while- als auch for-Schleifen, welche wir uns beide im Folgenden genauer ansehen werden (vgl. Listing \ref{refwhile} und \ref{reffor}). Schleifen bestehen aus einer Anweisung und einem Kontrollblock, welcher solange durchlaufen wird, bis die Anweisung oder ein Abbruchkriterium erfüllt wurde. Schleifen, die niemals ein Abbruchkriterium erfüllen und so endlos durchlaufen werden, heißen Endlosschleifen. Diese führen dazu, dass der Interpreter irgendwann aufgibt und abbricht.

\begin{lstlisting}[caption={While-Schleife},label=refwhile]
// While Schleife
while Bedingung:
	Anweisungsblock
	if Bedingung:
		Anweisungsblock
		continue
	if Bedingung:
		Anweisungsblock
		break
	Anweisungsblock

\end{lstlisting}

\begin{lstlisting}[caption={For-Schleife},label=reffor]
// For Schleife
for Variable in Objekt:
	Anweisungsblock
	if Bedingung:
		Anweisungsblock
		continue
	Anweisungsblock
	if Bedingung:
		Anweisungsblock
		break
	Anweisungsblock

\end{lstlisting}

\textbf{Enums}

Enums dienen in den objektorientierten Programmiersprachen zur Aufzählung von Ausdrücken einer endlichen Menge. So werden zum Beispiel Jahreszeiten, Monate oder Farben oft als Enums umgesetzt (vgl. Listing \ref{refenum}). 

\begin{lstlisting}[caption={Enums},label=refenum]
from enum import Enum
class Color(Enum):
	RED = 1
	GREEN = 2
	BLUE = 3

\end{lstlisting}
