\section{Kontrollstrukturen}

Die Kontrollstrukturen in Python haben einen formalen Unterschied zu Java oder C++, funktional sind sie identisch. In Python werden keine geschweiften Klammern benötigt, um die Blöcke der einzelnen Abfragen abzugrenzen. Dazu genügt das Einrücken der Anweisung. Dies gilt sowohl für if-then-else-Bedingungen und Conditional Expressions, als auch für Schleifen. Im Folgenden schauen wir uns die einzelnen Strukturen im Detail und mit Beispielen an.

\textbf{If-then-else}

Die if-then-else-Struktur ermöglicht es - wie wir es bereits kennen - simple wenn-dann Abfragen zu tätigen. \textit{(Beispiel für einmal else, dann if)}

Mehrere Abfragemöglichkeiten werden mit elif markiert. \textit{(Beispiel)}

\begin{lstlisting}
if statement1:
	print("Fall 1")
elif statement2:
	print("Fall 2")
else:
	print("Fall 3")

\end{lstlisting}

\textbf{Conditional Expressions}

Die Conditional Expressions (= bedingte Ausdrücke) stellen eine kompaktere Schreibweise als if-then-else-Bedinungen dar.  \textit{(Beipspiel)}

\begin{lstlisting}
// Klassisches If-Else
if wort == "start":
	x = "los"
else:
	x = halt"
	
// If-Else als Conditional Expression
x = ("los" if wort == "start" else "halt")

\end{lstlisting}

\textbf{Schleifen}

Python hat sowohl while- als auch for-Schleifen, welche wir uns beide im Folgenden genauer ansehen werden. Schleifen bestehen aus einer Anweisung und einem Kontrollblock, welcher solange durchlaufen wird, bis die Anweisung oder ein Abbruchkriterium erfüllt wurde. Schleifen die niemals ein Abbruchkriterium erfüllen und so endlos durchlaufen werden heißen Endlosschleifen. Diese führen dazu, dass der Compiler irgendwann den Geist aufgibt und abbricht. \textit{(Beispiele für while und for)}

\begin{lstlisting}
// While Schleife
while Bedingung:
	Anweisungsblock
	if Bedingung:
		Anweisungsblock
		continue
	if Bedingung:
		Anweisungsblock
		break
	Anweisungsblock

\end{lstlisting}

\begin{lstlisting}
// For Schleife
for Variable in Objekt:
	Anweisungsblock
	if Bedingung:
		Anweisungsblock
		continue
	Anweisungsblock
	if Bedingung:
		Anweisungsblock
		break
	Anweisungsblock

\end{lstlisting}

\textbf{Enums}

Enums dienen in den objektorientierten Programmiersprachen zur Aufzählung von Ausdrücken einer endlichen Menge. So werden zum Beispiel Jahreszeiten, Monate oder Farben oft als Enums umgesetzt. \textit{(Beispiel(e))}

\begin{lstlisting}
from enum import Enum
class Color(Enum):
	RED = 1
	GREEN = 2
	BLUE = 3

\end{lstlisting}
