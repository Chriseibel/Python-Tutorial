\section{Grundlagen Zusammenfassung} 
\label{grundlagenzs:sec:Zusammenfassung}

In diesem Kapitel hat der Leser seine ersten Schritte mit der Sprache Python gemacht. 
Es wurde grundlegend vorgestellt, wobei es sich bei Python überhaupt handelt und wie die Programmiersprache installiert wird. 
Es wurde der Python Interpreter vorgestellt, einer einfachen Konsolenanwendung zur Ausführung von Python Code. 
Weiterhin wurden Kommentare und die Blockstruktur eingeführt. 
Um das in diesem Kapitel erlangte Wissen anzuwenden, wurde ein klassisches Hello-World! Programm geschrieben. 
Um einen guten Einstieg in die Programmierung zu bekommen, war ein erster Überblick über die verschiedenen IDE's nötig. In dieser Übersicht sollte klar geworden sein, dass es eine Vielzahl von Entwicklungsumgebungen gibt, mit deren Hilfe man Python programmieren kann.
Anschließend wurden die unterschiedlichen einfachen Datentypen betrachtet. Dabei wurden neben den Zahlenwerten, Boolean- und Stringvariablen auch die Aufzählungen mit der Hilfe von ENUMs gezeigt. Dem Leser wurde ebenfalls gezeigt, dass es ein neutrales Element gibt, das an verschiedenen Stellen zum Einsatz gebracht werden kann. Im Zusammenhang mit einfachen Datentypen wurde auch eine Besonderheit von Python aufgezeigt, nämlich dass eine Variable ein Objekt referenziert und damit keinem festen Typ zugeordnet wird.
Im Weiteren wurde der Umgang mit Abfragen gezeigt. Zum einen wurde dabei betrachtet wie man if-else Anweisungen einsetzt und wie alternativ dazu Conditional Expressions verwendet werden können. Zum anderen fiel der Blick auf die Benutzung von Schleifen in der Python-Programmierung. Zum Abschluss des Themas Kontrollstrukturen wurde noch kurz die allgemeine Übersicht zu den gängigen logischen Operatoren veranschaulicht.
